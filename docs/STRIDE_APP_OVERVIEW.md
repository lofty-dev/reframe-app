# Stride アプリ概要

AIに渡すための情報まとめ。

---

## アプリの基本情報

- **名称**: Stride
- **種別**: PWA（Progressive Web App）、スマホへのインストール対応
- **対象**: 日本語ユーザー向けメンタルヘルスサポートアプリ
- **データ保存先**: ローカルの `localStorage`（サーバー送信なし、完全オフライン）
- **技術スタック**: React 19 + Vite、`@tabler/icons-react`、`@dnd-kit`（ドラッグ&ドロップ）

---

## アプリの目的

精神科・心療内科に通院中または心理的な困難を抱えるユーザーが、  
日常のセルフケアを継続し、支援者（主治医・カウンセラーなど）とのコミュニケーションをスムーズにするためのツール。

CBT（認知行動療法）の手法をベースに、「記録 → 対処 → 振り返り」のサイクルを自分でできるように設計されている。

---

## 機能一覧

### 1. チェックイン（毎日の状態記録）

- 気分を 1〜10 のスライダーで記録
- 体調・睡眠時間を選択肢から記録
- ひとことメモ（任意）
- 過去のチェックインを編集可能
- 記録は**気分グラフ**（折れ線・棒グラフ、14日/30日）と**週次レポート**で振り返れる

### 2. ストレス記録

- 気になった出来事・ストレスを記録
- カテゴリ（人間関係・仕事・健康・お金・将来・家族・その他）
- 強度（軽い・中・重い）
- 浮かんだ最初の考えを記録
- 記録後に以下のアプローチを選べる：
  - 認知再構成（3コラムまたは7コラム）
  - 問題解決技法
  - コーピング実践

### 3. 認知再構成（CBT）

**3コラム版**（シンプル）：
1. 状況
2. 感情（名前 + 強さ）
3. 自動思考 → 認知のクセの選択

**7コラム版**（詳細）：
1. 状況
2. 感情
3. 自動思考
4. 認知のクセ（10種類から選択）
5. 根拠（そう思う理由）
6. 反証（別の見方）
7. バランス思考
8. 感情の変化（再評価）

認知のクセの選択肢：白黒思考、べき思考、読心術、破局視、感情的決めつけ、過度の一般化、拡大視、縮小視、自己関連付け、その他。

### 4. 問題解決技法（PS）

7ステップで構造的に問題を整理する：
1. 問題状況の記述
2. 問題の細分化
3. 取り組む問題の選択
4. 解決のメリットを考える
5. 解決策を出す
6. 解決策を選ぶ
7. 実行タイミングを決める

後日、結果と気づきを振り返りとして記録できる。

### 5. コーピングリスト

- 自分なりの対処法（コーピング）を事前に登録
- 難易度・効果を評価してソート可能
- ストレス記録からアプローチとして使うか、一覧から直接実践できる
- 実践後に効果を記録して振り返れる

### 6. できたことログ（Achievement）

- 毎日の小さな達成・良かったことを自由記録
- カレンダーで過去の記録を振り返り
- 連続記録日数でモチベーション維持

### 7. マインドフルネスタイマー

- 1・3・5・7・10分から時間を選んでタイマースタート
- ホワイトノイズBGM付き
- 終了時にベル音で通知

### 8. クライシスプラン

3段階で自分の状態・対処法・連絡先を事前に整理：
- **SAFE**：安定しているときの状態
- **CAUTION**：調子が崩れるサインや引き金
- **CRISIS**：危機状態のサインと連絡先

PDF出力して診察に持参する機能あり。

### 9. 伝えたいことメモ

- 診察・面談で伝えたいことを事前に整理
- 誰に伝えるか（登録済みの支援者）を選んで紐付け
- 診察中に伝えた項目にチェック
- 伝えた後に「相手から言われたこと」を記録
- 全チェック完了で「完了」になり診察記録に自動反映

### 10. Bridge Session

- 診察・面談の場でアプリを支援者に見せながら話すための機能
- 表示する情報（気分グラフ・睡眠・ストレス記録・伝えたいことメモなど）をカスタマイズ
- セッション中にその場でメモを取れる
- メモは伝えたいことメモの回答欄に自動記録される

### 11. 診察等の記録

- 支援者（主治医・精神保健福祉士・カウンセラーなど）ごとにタブで記録を管理
- 伝えたいことメモの完了データが自動で反映される
- やりとりを時系列で振り返れる

### 12. ノート（自由メモ）

- タイトル・本文・日付を自由に記録できるメモ機能

## ナビゲーション構造

BottomNavで4タブに分かれている：

| タブ | 内容 |
|------|------|
| ホーム | チェックイン、マインドフルネス、できたことログ、使い方ガイド |
| 記録 | ストレス記録一覧、チェックイン履歴（グラフ・レポート・一覧） |
| ツール | 認知再構成、問題解決技法、コーピングリスト、ノート |
| 医療 | 伝えたいことメモ、Bridge Session、診察等の記録、クライシスプラン |

設定（テーマ切替・データエクスポート/インポート・プライバシーポリシー）はホームまたはBottomNavのメニューからアクセス。

---

## その他の特徴

- **テーマ**: ダーク／ライトモード切替対応（デフォルトはダーク）
- **データ管理**: JSONエクスポート・インポート対応（端末間移行に使用）
- **PDF出力**: クライシスプランとストレス記録・コーピングリストがPDF出力対応
- **オンボーディング**: 初回起動時にスライド形式で機能紹介、プライバシーポリシー同意フロー
- **お知らせ**: アプリ内でアップデート情報を表示
- **問い合わせ**: LINEでの問い合わせ機能

## コードの構成

```
src/
  App.jsx          # すべてのページ・ロジックを含むメインファイル（333KB超）
  constants.js     # 定数・CBTステップ定義・認知のクセ・カラーテーマなど
  storage.js       # localStorage のread/writeヘルパー関数
  styles.js        # 共通スタイルオブジェクト
  components/
    BottomNav.jsx        # 下部ナビゲーション
    DateSelector.jsx     # 日付選択コンポーネント
    EmotionInput.jsx     # 感情入力・自動思考入力
    SortablePersonItem.jsx  # dnd-kitで並び替え可能な人物リスト
    PageErrorBoundary.jsx   # ページ別エラーバウンダリ
```

ビュー遷移はすべて `view` stateで管理（React Routerは不使用）。  
データはすべてlocalStorageに永続化。外部APIへの通信は一切なし。
